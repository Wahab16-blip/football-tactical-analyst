# ⚽ Football Tactical Analyst

An Ai-powered tactical analysis tool for football coaches.
Built with Python, Claude AI and Streamlit.

## Features
- Squad management with position-specific ability ratings
- AI-generated formation recommendations based on player data
- Live weather integration for match conditions analysis
- Detailed tactical reports with formation visualisation
- Save, compare and download reports

## Tech Stack
- **AI:** Anthropic Claude API, RAG pipeline, Multi-tool agents
- **Embeddings:** sentence-transformers, cosine similarity
- **Web:** Streamlit
- **APIs:** wttr.in weather API
- **Storage:** JSON persistence
